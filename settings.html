Settings

Use this page to configure the Together API key and set up passkey protection.

  1. Enter your Together API key below and press Save API Key.
  2. If no passkey exists, the browser will ask you to create one. This passkey is used to encrypt your key.
  3. After saving, use Decrypt with passkey to authenticate and confirm the key was stored correctly.
  4. To start over at any time, click Reset API Key which deletes the stored key and passkey information.
