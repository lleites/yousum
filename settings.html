Settings

Use this page to configure the Together API key and set up PIN protection. If your browser supports passkeys, a fingerprint or device authentication may also be requested.

  1. Enter your Together API key and choose a PIN, then press Save API Key.
  2. If your browser supports passkeys, it may prompt you to create one for biometric authentication.
  3. After saving, use Decrypt stored key to authenticate and confirm the key was stored correctly.
  4. To start over at any time, click Reset API Key which deletes the stored key and passkey information.
